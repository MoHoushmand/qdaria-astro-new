Method 2: Clear Browser Cache & HSTS

Google Chrome:
  1. Type chrome://settings/security in address bar
  2. Scroll to "Manage certificates" or "Security"
  3. Clear HSTS for qdaria.com
  4. Or type chrome://net-internals/#hsts
  5. Under "Delete domain security policies", enter: qdaria.com
  6. Click "Delete"
  7. Clear cache: Ctrl+Shift+Delete (Windows) or Cmd+Shift+Delete (Mac)
Firefox:
  1. Type about:config in address bar
  2. Search for network.stricttransportsecurity.preloadlist
  3. Set to false temporarily
  4. Clear cache: Ctrl+Shift+Delete (Windows) or Cmd+Shift+Delete (Mac)
  5. Visit the site, then set back to true
Safari:
  1. Safari → Preferences → Privacy
  2. Click "Manage Website Data"
  3. Search for "qdaria"
  4. Remove all entries
  5. Clear cache: Cmd+Option+E

Method 3: Change DNS Servers

Use public DNS servers for better reliability:

Windows:
  1. Open Control Panel → Network and Internet → Network Connections
  2. Right-click your connection → Properties
  3. Select "Internet Protocol Version 4 (TCP/IPv4)" → Properties
  4. Choose "Use the following DNS server addresses"
  5. Enter:
    Preferred: 8.8.8.8 Alternate: 8.8.4.4
    Or for Cloudflare:
    Preferred: 1.1.1.1 Alternate: 1.0.0.1
  6. Click OK and restart browser
macOS:
  1. System Preferences → Network
  2. Select your connection → Advanced → DNS
  3. Click + and add:
    8.8.8.8 8.8.4.4
  4. Click OK → Apply
Quick DNS Flush Commands:
# Windows ipconfig /flushdns # macOS sudo dscacheutil -flushcache # Linux sudo systemd-resolve --flush-caches
